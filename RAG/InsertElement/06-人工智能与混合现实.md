# 人工智能与混合现实 (Artificial Intelligence & Mixed Reality) 组件

---

## 人工智能 (Artificial Intelligence)

> **注意**：以下 AI 组件均为"高级功能"，需要 Power Apps Premium 或 Per App Plan 许可证。

### 1. Business Card Reader (名片阅读器 - 高级功能)

**操作方式**：插入 → 人工智能 → Business card reader

拍摄并自动识别名片上的联系人信息。

| 属性 | 类型 | 说明 |
|------|------|------|
| Image | Image | 输入的名片图像 |
| BusinessCardName | String | 识别出的姓名（输出） |
| BusinessCardFirstName | String | 识别出的名字（输出） |
| BusinessCardLastName | String | 识别出的姓氏（输出） |
| BusinessCardEmail | String | 识别出的邮箱（输出） |
| BusinessCardPhone | String | 识别出的电话（输出） |
| BusinessCardCompany | String | 识别出的公司名称（输出） |
| BusinessCardJobTitle | String | 识别出的职位（输出） |
| BusinessCardAddress | String | 识别出的地址（输出） |
| BusinessCardWebsite | String | 识别出的网站（输出） |
| BusinessCardDepartment | String | 识别出的部门（输出） |
| OnChange | Action | 识别完成时的行为 |
| BorderColor | Color | 边框颜色 |
| Fill | Color | 背景色 |
| Height | Number | 高度 |
| Width | Number | 宽度 |

## 混合现实 (Mixed Reality)

> **注意**：混合现实组件需要设备支持（如 HoloLens、iOS/Android 设备）并安装 Power Apps Mobile。

### 1. View in MR (在混合现实中查看)

**操作方式**：插入 → 人工智能 → 混合现实 → View in MR

在混合现实环境中查看 3D 内容。

| 属性 | 类型 | 说明 |
|------|------|------|
| Source | String | 3D 模型来源 URL（GLB/GLTF 格式） |
| PositionX | Number | 模型在 MR 空间中的 X 位置 |
| PositionY | Number | 模型在 MR 空间中的 Y 位置 |
| PositionZ | Number | 模型在 MR 空间中的 Z 位置 |
| Scale | Number | 模型缩放比例 |
| RotationX | Number | 绕 X 轴旋转角度 |
| RotationY | Number | 绕 Y 轴旋转角度 |
| RotationZ | Number | 绕 Z 轴旋转角度 |
| OnSelect | Action | 点击时的行为 |
| Height | Number | 高度 |
| Width | Number | 宽度 |

---

### 2. View Shape in MR (在混合现实中查看形状)

**操作方式**：插入 → 人工智能 → 混合现实 → View shape in MR

在混合现实中查看基本形状。

| 属性 | 类型 | 说明 |
|------|------|------|
| ShapeType | Enum | 形状类型: `Box` / `Sphere` / `Cylinder` / `Cone` / `Torus` |
| Color | Color | 形状颜色 |
| Scale | Number | 缩放比例 |
| PositionX/Y/Z | Number | MR 空间位置坐标 |
| RotationX/Y/Z | Number | MR 空间旋转角度 |
| OnSelect | Action | 点击时的行为 |
| Height | Number | 高度 |
| Width | Number | 宽度 |

---

### 3. Measure in MR (在混合现实中测量)

**操作方式**：插入 → 人工智能 → 混合现实 → Measure in MR

在混合现实中进行距离/面积测量。

| 属性 | 类型 | 说明 |
|------|------|------|
| MeasurementType | Enum | 测量类型: `Distance` / `Area` |
| MeasurementUnit | Enum | 单位: `Meters` / `Centimeters` / `Feet` / `Inches` |
| MeasuredValue | Number | 测量结果数值（输出） |
| MeasuredValueUnit | String | 测量结果单位（输出） |
| OnMeasurementComplete | Action | 测量完成时的行为 |
| Height | Number | 高度 |
| Width | Number | 宽度 |
