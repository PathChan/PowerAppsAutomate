# 显示 (Display) 组件

## 1. Text (文本)

**操作方式**：插入 → 显示 → Text

纯文本显示控件。

| 属性 | 类型 | 说明 |
|------|------|------|
| Text | String | 显示的文本内容（支持 HTML 标记） |
| AutoHeight | Boolean | 是否根据内容自动调整高度 |
| Overflow | Enum | 文本溢出处理: `Overflow.Hidden` / `Overflow.Scroll` / `Overflow.Auto` |
| LineLimit | Number | 最大行数限制 |
| Wrap | Boolean | 是否自动换行 |
| Role | Enum | 语义角色: `Default` / `Heading1` ~ `Heading6` / `Body` |
| Color | Color | 文字颜色 |
| Font | Enum | 字体 |
| FontSize | Number | 字号 |
| FontWeight | Enum | 字重 |
| Italic | Boolean | 斜体 |
| Underline | Boolean | 下划线 |
| Strikethrough | Boolean | 删除线 |
| Align | Enum | 水平对齐 |
| VerticalAlign | Enum | 垂直对齐 |
| PaddingLeft/Right/Top/Bottom | Number | 内边距 |
| DropShadow | Enum | 阴影效果 |
| BorderColor | Color | 边框颜色 |
| Fill | Color | 背景色 |

---

## 2. Information Button (信息按钮)

**操作方式**：插入 → 显示 → Information button

带信息图标（ℹ）的按钮，点击通常弹出提示或帮助信息。

| 属性 | 类型 | 说明 |
|------|------|------|
| Text | String | 按钮上的文本 |
| OnSelect | Action | 点击时的行为（通常弹出对话框或导航） |
| Color | Color | 图标颜色 |
| Fill | Color | 背景色 |
| Radius | Number | 圆角半径 |
| Size | Number | 图标大小 |
| DisplayMode | Enum | 交互模式 |

---

## 3. Icon (图标)

**操作方式**：插入 → 显示 → Icon

预定义图标控件。

| 属性 | 类型 | 说明 |
|------|------|------|
| Icon | Enum | 图标类型: `Icon.Add` / `Icon.Delete` / `Icon.Edit` / `Icon.Search` 等数百种 |
| Color | Color | 图标颜色 |
| Size | Number | 图标尺寸 |
| Rotation | Number | 旋转角度 |
| OnSelect | Action | 点击图标时的行为 |
| DisplayMode | Enum | 交互模式 |
| Fill | Color | 背景色（透明背景时无效） |

---

## 4. Avatar (头像)

**操作方式**：插入 → 显示 → Avatar

用户头像显示控件，支持图片或首字母。

| 属性 | 类型 | 说明 |
|------|------|------|
| Image | Image | 头像图片 |
| Text | String | 无图片时显示的文本（通常为首字母） |
| Shape | Enum | 形状: `AvatarShape.Circular` / `AvatarShape.Square` |
| Size | Number | 头像尺寸 |
| Color | Color | 文字颜色 |
| Fill | Color | 背景色 |
| OnSelect | Action | 点击头像时的行为 |

---

## 5. Badge (徽章)

**操作方式**：插入 → 显示 → Badge

角标/徽章控件，用于显示计数或状态标记。

| 属性 | 类型 | 说明 |
|------|------|------|
| Text | String | 徽章内显示的文本 |
| Color | Color | 文字颜色 |
| Fill | Color | 背景色 |
| Size | Number | 尺寸 |
| Shape | Enum | 形状: `BadgeShape.Circular` / `Rounded` / `Square` |
| Visible | Boolean | 是否可见 |
| Position | Enum | 位置: `TopLeft` / `TopRight` / `BottomLeft` / `BottomRight` |

---

## 6. Card (卡片)

**操作方式**：插入 → 显示 → Card

内容卡片容器。

| 属性 | 类型 | 说明 |
|------|------|------|
| Items | Table | 卡片内子组件数据 |
| Title | String | 卡片标题 |
| Subtitle | String | 卡片副标题 |
| Image | Image | 卡片图片 |
| ImagePosition | Enum | 图片位置: `Above` / `Below` / `Left` / `Right` |
| DropShadow | Enum | 阴影效果 |
| Radius | Number | 圆角半径 |
| BorderColor | Color | 边框颜色 |
| Fill | Color | 背景色 |
| OnSelect | Action | 点击卡片时的行为 |
| Height | Number | 高度 |
| Width | Number | 宽度 |

---

## 7. Progress Bar (进度条)

**操作方式**：插入 → 显示 → Progress Bar

显示任务进度的条形控件。

| 属性 | 类型 | 说明 |
|------|------|------|
| Value | Number | 当前进度值（0-100） |
| Min | Number | 最小值（默认为 0） |
| Max | Number | 最大值（默认为 100） |
| ShowPercent | Boolean | 是否显示百分比文本 |
| Color | Color | 进度条填充色 |
| Fill | Color | 轨道背景色 |
| Height | Number | 进度条厚度 |
| Label | String | 标签文本 |
| Thickness | Number | 进度条粗细 |
| Variant | Enum | 样式变体: `Determinate` / `Indeterminate`（不确定进度时使用） |

---

## 8. Spinner (加载等待器)

**操作方式**：插入 → 显示 → Spinner

加载中的旋转动画指示器。

| 属性 | 类型 | 说明 |
|------|------|------|
| Size | Number | 尺寸（Small / Medium / Large） |
| Color | Color | 旋转动画颜色 |
| Fill | Color | 背景色 |
| Visible | Boolean | 是否可见 |
| Label | String | 加载提示文本 |

---

## 9. Link (链接)

**操作方式**：插入 → 显示 → Link

超链接文本控件。

| 属性 | 类型 | 说明 |
|------|------|------|
| Text | String | 链接显示的文本 |
| Url | String | 点击时导航的 URL |
| OpenInNewTab | Boolean | 是否在新标签页打开 |
| Underline | Boolean | 是否显示下划线 |
| Color | Color | 链接颜色 |
| HoverColor | Color | 悬停时的颜色 |
| FontSize | Number | 字号 |
| FontWeight | Enum | 字重 |
| OnSelect | Action | 点击链接时的行为 |
| DisplayMode | Enum | 交互模式 |

---

## 10. Timer (定时器)

**操作方式**：插入 → 显示 → Timer

定时触发动作的控件（不可见）。

| 属性 | 类型 | 说明 |
|------|------|------|
| Duration | Number | 定时间隔（毫秒） |
| AutoStart | Boolean | 是否自动开始计时 |
| Repeat | Boolean | 是否重复触发 |
| Started | Boolean | 计时器是否已开始（只读） |
| RemainingTime | Number | 剩余时间（毫秒，只读） |
| OnTimerStart | Action | 计时开始时的行为 |
| OnTimerEnd | Action | 计时结束时的行为 |
| Visible | Boolean | 是否可见（通常设为 false） |
| DisplayMode | Enum | 交互模式 |

---

**操作方式**：插入 → 显示 → HTML text

| Width | Number | 宽度 |
