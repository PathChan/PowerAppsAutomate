# 输入 (Input) 组件

## 1. Button (按钮)

**操作方式**：插入 → 输入 → Button；或经典分类下也有 Button

用于触发操作的按钮。

| 属性 | 类型 | 说明 |
|------|------|------|
| Text | String | 按钮上显示的文本 |
| OnSelect | Action | 点击按钮时执行的动作 |
| Icon | Icon | 按钮图标 |
| IconPosition | Enum | 图标位置: `IconPosition.Left` / `Right` / `Top` / `Bottom` |
| Radius | Number | 圆角半径 |
| DropShadow | Enum | 阴影效果 |
| BorderColor | Color | 边框颜色 |
| Fill | Color | 背景色 |
| Color | Color | 文字颜色 |
| DisplayMode | Enum | 模式: `Edit` / `View` / `Disabled` |
| FontWeight | Enum | 字重: `Bold` / `Semibold` / `Normal` |
| DisabledColor | Color | 禁用时的文字颜色 |
| DisabledFill | Color | 禁用时的背景色 |
| HoverColor | Color | 悬停时的文字颜色 |
| HoverFill | Color | 悬停时的背景色 |
| PressedColor | Color | 按下时的文字颜色 |
| PressedFill | Color | 按下时的背景色 |
| Loading | Boolean | 是否显示加载状态 |
| OnSelect | Action | 点击时行为 |

## 2. Text Input (文本输入)

**操作方式**：插入 → 输入 → Text input

单行或多行文本输入框。

| 属性 | 类型 | 说明 |
|------|------|------|
| Default | String | 默认值 |

| HintText | String | 占位符提示文本 |
| Mode | Enum | 模式: `SingleLine` / `MultiLine` |
| MaxLength | Number | 最大字符数 |
| Format | Enum | 格式: `Text` / `Number` / `Email` / `Phone` / `Password` |
| Align | Enum | 文本对齐方式 |
| ClearButton | Boolean | 是否显示清除按钮 |
| DisplayMode | Enum | 交互模式 |
| OnChange | Action | 值变化时的行为 |
| OnSelect | Action | 点击时的行为 |
| BorderColor | Color | 边框颜色 |
| Fill | Color | 背景色 |
| Color | Color | 文字颜色 |
| FontSize | Number | 字号 |
| Underline | Boolean | 下划线 |
| EnableSpellCheck | Boolean | 是否启用拼写检查 |
| VirtualKeyboardMode | Enum | 虚拟键盘模式 |
