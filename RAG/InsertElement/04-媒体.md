# 媒体 (Media) 组件

## 1. Image (图像)

**操作方式**：插入 → 媒体 → Image

显示图片的控件。

| 属性 | 类型 | 说明 |
|------|------|------|
| Image | Image | 图片来源 URL 或二进制数据 |
| AlternateText | String | 图片无法加载时的替代文本 |
| ImagePosition | Enum | 图片填充方式: `Fill` / `Fit` / `Stretch` / `Tile` / `Center` |
| Radius | Number | 圆角半径 |
| DropShadow | Enum | 阴影效果 |
| BorderColor | Color | 边框颜色 |
| BorderStyle | Enum | 边框样式 |
| BorderThickness | Number | 边框厚度 |
| OnSelect | Action | 点击图片时的行为 |
| Tooltip | String | 悬停提示 |
| Rotation | Number | 旋转角度 |
| TransformedImage | Image | 变换后的图像（只读） |
| ShowLoadingImage | Boolean | 加载中是否显示占位图 |

## 2. Camera (相机)

**操作方式**：插入 → 媒体 → Camera

调用设备相机拍照或扫描。

| 属性 | 类型 | 说明 |
|------|------|------|
| Camera | Boolean | 是否启用相机 |
| StreamRate | Number | 视频流刷新率（毫秒） |
| Photo | Image | 最近拍摄的照片（输出） |
| Brightness | Number | 亮度调节（百分比） |
| Contrast | Number | 对比度调节 |
| OnStream | Action | 视频流每一帧到达时的行为 |
| OnSelect | Action | 点击拍照时的行为 |
| ShowControls | Boolean | 是否显示控制按钮 |
| FrontCamera | Boolean | 是否使用前置摄像头 |
| Height | Number | 高度 |
| Width | Number | 宽度 |
| BorderColor | Color | 边框颜色 |
