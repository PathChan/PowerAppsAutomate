# 经典 (Classic) 组件

## 1. Text Label (文本标签)

**操作方式**：插入 → 经典 → Text label

纯文本显示控件（经典版）。

| 属性 | 类型 | 说明 |
|------|------|------|
| Text | String | 显示的文本 |
| AutoHeight | Boolean | 是否自动调整高度 |
| Overflow | Enum | 溢出模式: `Overflow.Hidden` / `Overflow.Scroll` |
| Wrap | Boolean | 是否自动换行 |
| Live | Enum | 辅助功能实时区域: `Off` / `Polite` / `Assertive` |
| Role | Enum | 语义角色 |
| Color | Color | 文字颜色 |
| Font | Enum | 字体 |
| FontSize | Number | 字号 |
| FontWeight | Enum | 字重 |
| Italic | Boolean | 斜体 |
| Underline | Boolean | 下划线 |
| Strikethrough | Boolean | 删除线 |
| Align | Enum | 水平对齐 |
| VerticalAlign | Enum | 垂直对齐 |
| PaddingLeft/Right/Top/Bottom | Number | 内边距 |
| BorderColor | Color | 边框颜色 |
| BorderStyle | Enum | 边框样式 |
| BorderThickness | Number | 边框厚度 |
| Fill | Color | 背景色 |

## 6. Edit Form (编辑表单)

**操作方式**：插入 → 经典 → Edit form

|------|------|------|
| DataSource | DataSource | 绑定的数据源 |

| Mode | Enum | 表单模式: `FormMode.New` / `FormMode.Edit` / `FormMode.View` |
| DefaultMode | Enum | 默认打开时的模式 |
| Fields | Table | 字段集合（添加字段后自动生成数据卡片） |
| Selected | Record | 当前选中的数据卡片 |
| Valid | Boolean | 表单数据是否全部有效（只读） |
| Unsaved | Boolean | 是否有未保存的更改（只读） |
| OnNew | Action | 进入新建模式时的行为 |
| OnEdit | Action | 进入编辑模式时的行为 |
| OnView | Action | 进入查看模式时的行为 |
| BorderColor | Color | 边框颜色 |
| Fill | Color | 背景色 |
