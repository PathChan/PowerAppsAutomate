# 图表与形状 (Charts & Shapes) 组件

---

## 图表 (Charts)

### 1. Column Chart (柱状图)

**操作方式**：插入 → 图表与形状 → 图表 → Column chart

垂直柱状图，用于比较不同类别的数据。

| 属性 | 类型 | 说明 |
|------|------|------|
| Items | Table | 图表数据源 |
| XAxisLabel | String | X 轴标签文本 |
| YAxisLabel | String | Y 轴标签文本 |
| XAxisWidth | Number | X 轴宽度 |
| YAxisWidth | Number | Y 轴宽度 |
| GridLine | Boolean | 是否显示网格线 |
| NumberOfSeries | Number | 数据系列数量 |
| Series1~SeriesN | Number | 各数据系列的值字段 |
| AxisMin | Number | Y 轴最小值 |
| AxisMax | Number | Y 轴最大值 |
| ShowLegend | Boolean | 是否显示图例 |
| LegendPosition | Enum | 图例位置 |
| BorderColor | Color | 边框颜色 |
| Fill | Color | 背景色 |
| Color | Color | 柱体颜色 |
| Width | Number | 宽度 |
| Height | Number | 高度 |

---

### 2. Pie Chart (饼图)

**操作方式**：插入 → 图表与形状 → 图表 → Pie chart

饼图/环形图，用于展示数据占比。

| 属性 | 类型 | 说明 |
|------|------|------|
| Items | Table | 图表数据源 |
| ValueField | String | 数值字段名 |
| LabelField | String | 标签字段名 |
| ColorField | String | 颜色字段名 |
| ShowLegend | Boolean | 是否显示图例 |
| LegendPosition | Enum | 图例位置 |
| Donut | Boolean | 是否为环形图 |
| InnerRadius | Number | 内环半径比例（环形图时） |
| BorderColor | Color | 边框颜色 |
| Fill | Color | 背景色 |
| Width | Number | 宽度 |
| Height | Number | 高度 |

---

### 3. Line Chart (折线图)

**操作方式**：插入 → 图表与形状 → 图表 → Line chart

折线图，用于展示数据趋势。

| 属性 | 类型 | 说明 |
|------|------|------|
| Items | Table | 图表数据源 |
| XAxisLabel | String | X 轴标签 |
| YAxisLabel | String | Y 轴标签 |
| NumberOfSeries | Number | 数据系列数量 |
| Series1~SeriesN | Number | 各数据系列的值字段 |
| AxisMin | Number | Y 轴最小值 |
| AxisMax | Number | Y 轴最大值 |
| ShowLegend | Boolean | 是否显示图例 |
| LegendPosition | Enum | 图例位置 |
| ShowMarkers | Boolean | 是否显示数据点标记 |
| GridLine | Boolean | 是否显示网格线 |
| BorderColor | Color | 边框颜色 |
| Fill | Color | 背景色 |
| Color | Color | 折线颜色 |
| Width | Number | 宽度 |
| Height | Number | 高度 |

---

### 4. Power BI Tile (Power BI 磁贴)

**操作方式**：插入 → 图表与形状 → 图表 → Power BI tile

嵌入 Power BI 报表中的磁贴/仪表盘。

| 属性 | 类型 | 说明 |
|------|------|------|
| TileUrl | String | Power BI 磁贴/报表的嵌入 URL |
| ShowFilter | Boolean | 是否显示筛选器 |
| ShowTitle | Boolean | 是否显示标题 |
| BorderColor | Color | 边框颜色 |
| Fill | Color | 背景色 |
| Height | Number | 高度 |
| Width | Number | 宽度 |

## 形状 (Shapes)

### 通用形状属性

所有形状组件共享以下属性：

| 属性 | 类型 | 说明 |
|------|------|------|
| Fill | Color | 形状填充颜色 |
| Color | Color | 形状轮廓/线条颜色 |
| BorderColor | Color | 边框颜色 |
| BorderStyle | Enum | 边框样式 |
| BorderThickness | Number | 边框厚度 |
| Width | Number | 形状宽度 |
| Height | Number | 形状高度 |
| X | Number | 水平位置 |
| Y | Number | 垂直位置 |
| Rotation | Number | 旋转角度（度） |
| OnSelect | Action | 点击形状时的行为 |
| Visible | Boolean | 是否可见 |
| DisplayMode | Enum | 交互模式 |
