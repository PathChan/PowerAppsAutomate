# 布局 (Layout) 组件

## 1. Container (容器)

**操作方式**：插入 → 布局 → Container / 水平容器 / 垂直容器 / 网格容器

Container 是一个通用容器，用于将多个组件分组并在逻辑上和视觉上组织在一起。

| 属性 | 类型 | 说明 |
|------|------|------|
| Items | Table | 容器内子组件的集合（动态容器） |
| Layout | Enum | 布局方向: `Layout.Vertical` / `Layout.Horizontal` |
| Wrap | Boolean | 是否允许子组件换行 |
| Gap | Number | 子组件之间的间距 |
| Padding | Number | 容器内边距 |
| BorderColor | Color | 边框颜色 |
| BorderStyle | Enum | 边框样式 |
| BorderThickness | Number | 边框厚度 |
| DropShadow | Enum | 阴影效果: `DropShadow.None` / `DropShadow.Small` / `DropShadow.Medium` / `DropShadow.Large` |
| Radius | Number | 圆角半径 |
| OnSelect | Action | 点击容器时的行为 |

---

## 2. Horizontal Container (水平容器)

**操作方式**：插入 → 布局 → Horizontal container

自动水平排列子组件的容器。

| 属性 | 类型 | 说明 |
|------|------|------|
| LayoutDirection | Enum | 方向固定为 `LayoutDirection.Horizontal` |
| JustifyContent | Enum | 水平对齐方式: `Start` / `End` / `Center` / `SpaceBetween` / `SpaceAround` |
| AlignItems | Enum | 垂直对齐方式: `Start` / `End` / `Center` / `Stretch` |
| Gap | Number | 子组件间距 |
| Fill | Color | 背景色 |
| Wrap | Boolean | 是否允许换行 |

---

## 3. Vertical Container (垂直容器)

**操作方式**：插入 → 布局 → Vertical container

自动垂直排列子组件的容器。

| 属性 | 类型 | 说明 |
|------|------|------|
| LayoutDirection | Enum | 方向固定为 `LayoutDirection.Vertical` |
| JustifyContent | Enum | 垂直对齐方式 |
| AlignItems | Enum | 水平对齐方式 |
| Gap | Number | 子组件间距 |
| Fill | Color | 背景色 |

---

## 4. Grid Container (网格容器)

**操作方式**：插入 → 布局 → Grid container

基于 CSS Grid 布局的容器，支持行列网格排列子组件。

| 属性 | 类型 | 说明 |
|------|------|------|
| Columns | Number | 网格列数 |
| Rows | Number | 网格行数 |
| ColumnGap | Number | 列间距 |
| RowGap | Number | 行间距 |
| ColumnSizes | String | 每列宽度比例，如 `"1fr 2fr 1fr"` |
| RowSizes | String | 每行高度比例 |
| Fill | Color | 背景色 |

---

## 5. Vertical Gallery (垂直库)

**操作方式**：插入 → 布局 → Vertical gallery

以垂直列表形式展示数据集合，每行显示一条记录。

| 属性 | 类型 | 说明 |
|------|------|------|
| Items | Table | 数据源，如 `Filter(Employees, Department = "IT")` |
| TemplateSize | Number | 每项模板的高度 |
| TemplatePadding | Number | 模板内边距 |
| WrapCount | Number | 每行显示的项数（多列布局） |
| ShowNavigation | Boolean | 是否显示分页导航按钮 |
| NavigationStep | Number | 翻页步长 |
| SnapToGrid | Boolean | 是否对齐到网格 |
| BorderColor | Color | 边框颜色 |
| Fill | Color | 背景色 |
| OnSelect | Action | 选中某项时的行为 |
| Default | Record | 默认选中项 |
| Selected | Record | 当前选中项（只读） |
| Selectable | Boolean | 是否可选中 |

---

## 7. Flexible Height Gallery (弹性高度库)

**操作方式**：插入 → 布局 → Flexible height gallery

每一项高度根据内容自动调整的垂直库。

| 属性 | 类型 | 说明 |
|------|------|------|
| Items | Table | 数据源 |
| TemplatePadding | Number | 模板内边距 |
| TemplateSize | Number | 最小模板高度 |
| AutoHeight | Boolean | 是否自动调整高度 |
| ShowNavigation | Boolean | 是否显示导航按钮 |
| OnSelect | Action | 选中项行为 |
| Default | Record | 默认选中项 |
| Selected | Record | 当前选中项 |

## 8. Header (页眉)

**操作方式**：插入 → 布局 → Header

预设样式的页面顶部标题栏。

| 属性 | 类型 | 说明 |
|------|------|------|
| Text | String | 标题文本 |
| Subtitle | String | 副标题文本 |
| BackgroundColor | Color | 背景色 |
| ForegroundColor | Color | 前景色/文字颜色 |
| Height | Number | 高度（固定） |
| OnSelect | Action | 点击页眉时的行为 |

---

## 9. Data Grid Preview (数据网格 - 预览)

**操作方式**：插入 → 布局 → Data grid Preview

以表格形式展示数据，支持排序、筛选、列调整。

| 属性 | 类型 | 说明 |
|------|------|------|
| Items | Table | 数据源 |
| Columns | Table | 列定义集合 |
| SortColumn | String | 当前排序列名称 |
| SortDirection | Enum | 排序方向: `SortDirection.Ascending` / `Descending` |
| SelectedItems | Table | 选中的行集合 |
| MultiSelect | Boolean | 是否允许多选 |
| OnSelect | Action | 选择行时的行为 |

---

**操作方式**：插入 → 布局 → Table Preview

| 属性 | 类型 | 说明 |
|------|------|------|
| Items | Table | 数据源 |
| Columns | Table | 列定义 |
| SelectedItems | Table | 选中行集合 |

---

## 11. Form (表单)

**操作方式**：插入 → 布局 → Form

数据输入表单容器，与数据源绑定后自动生成字段。

| 属性 | 类型 | 说明 |
|------|------|------|
| DataSource | DataSource | 绑定的数据源（如 SharePoint 列表、Dataverse 表） |
| Item | Record | 当前表单编辑/查看的记录 |
| Mode | Enum | 表单模式: `FormMode.New` / `FormMode.Edit` / `FormMode.View` |
| DefaultMode | Enum | 默认模式 |
| Fields | Table | 表单中的字段集合 |
| Selected | Record | 选中的数据卡片 |
| OnNew | Action | 新建模式触发时的行为 |
| OnEdit | Action | 编辑模式触发时的行为 |
| OnView | Action | 查看模式触发时的行为 |
| Valid | Boolean | 表单数据是否有效（只读） |
| Unsaved | Boolean | 是否有未保存更改（只读） |
| FocusedField | String | 当前聚焦的字段名 |
| BorderColor | Color | 边框颜色 |
| Fill | Color | 背景色 |
