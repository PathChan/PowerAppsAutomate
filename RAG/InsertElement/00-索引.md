# PowerApps 可插入组件 — 分类索引

## 通用操作方式

所有 PowerApps 组件的插入操作方式基本一致：

1. **通过插入面板**：在 Power Apps Studio 左侧依次点击「插入」→ 展开对应分类 → 点击或拖拽组件到画布
2. **通过公式栏**：在公式栏中使用 `Insert()` 函数（部分组件支持）
3. **通过复制粘贴**：选中已有组件，按 `Ctrl+C` / `Ctrl+V` 复制
4. **通过代码/脚本**：在 PCF / Connector 中通过代码动态创建

## 通用属性说明

以下属性几乎适用于所有组件（不重复列举）：

| 属性 | 类型 | 说明 |
|------|------|------|
| X, Y | Number | 组件在画布上的左上角坐标位置 |
| Width, Height | Number | 组件的宽度和高度 |
| Visible | Boolean | 是否可见 (`true`/`false`) |
| DisplayMode | Enum | 显示模式: `DisplayMode.Edit` / `DisplayMode.View` / `DisplayMode.Disabled` |
| TabIndex | Number | Tab 键导航顺序 |
| Tooltip | String | 悬停提示文本 |
| FocusedBorderColor | Color | 焦点状态边框颜色 |
| FocusedBorderThickness | Number | 焦点状态边框厚度 |
| BorderColor, BorderStyle, BorderThickness | Color/Enum/Number | 边框颜色、样式（Solid/Dash/Dotted/None）、厚度 |
| Fill | Color | 背景填充色 |
| Color | Color | 前景/文字颜色 |
| Font, FontSize, FontWeight | Enum/Number/Enum | 字体、字号、字重 |
| Italic, Underline, Strikethrough | Boolean | 斜体、下划线、删除线 |
| Align, VerticalAlign | Enum | 水平对齐（Left/Center/Right/Justify）和垂直对齐（Top/Middle/Bottom） |
| PaddingLeft, PaddingRight, PaddingTop, PaddingBottom | Number | 内边距 |
| OnSelect | Action | 点击/选中时的行为 |
| OnVisible | Action | 组件变为可见时的行为 |
| OnHidden | Action | 组件隐藏时的行为 |
