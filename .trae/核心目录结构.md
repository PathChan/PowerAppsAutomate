## 注意 本文件将需要根据项目进度进行更新

### 核心目录结构

```
PowerfulApps/                   # 是项目的主文件夹，其他的文件夹都是开发时测试文件夹，后续的发布版只会发布这个文件夹
├── Browser/                    # 浏览器控制层
│   ├── core/                   # 核心：会话管理、事件系统、配置、视图、watchdog 基类
│   │   ├── session.py          # BrowserSession 主类
│   │   ├── session_manager.py  # CDP session 管理
│   │   ├── profile.py          # 浏览器配置
│   │   ├── events.py           # 事件定义
│   │   ├── views.py            # 视图/数据模型
│   │   ├── action_result.py    # 操作结果
│   │   └── watchdog_base.py    # watchdog 基类
│   ├── cdp/                    # CDP 直连工具
│   │   ├── studio_cdp.py       # PowerApps Studio iframe CDP 操作
│   │   ├── _cdp_timeout.py     # CDP 请求超时包装
│   │   └── moc_target_cache.py # UI 目标坐标缓存
│   ├── utils/                  # 工具函数
│   │   ├── moc_config.py       # 配置系统
│   │   ├── moc_utils.py        # 通用工具（日志、信号处理等）
│   │   ├── moc_observability.py# 可观测性/链路追踪
│   │   ├── moc_llm_messages.py # LLM 消息类型
│   │   └── python_highlights.py# 截图标注
│   ├── actor/                  # CDP 高级抽象层（Playwright 风格）
│   │   ├── element.py, mouse.py, page.py, utils.py
│   ├── demo/                   # Demo 面板注入
│   ├── cloud/                  # 云端浏览器 stub（已废弃）
│   └── watchdogs/              # 浏览器监控组件
│       ├── default_action_watchdog.py  # 默认动作处理
│       ├── dom_watchdog.py             # DOM 树管理
│       ├── local_browser_watchdog.py   # 本地浏览器生命周期
│       ├── screenshot_watchdog.py      # 截图
│       ├── crash_watchdog.py           # 崩溃检测
│       ├── aboutblank_watchdog.py      # about:blank 处理
│       ├── captcha_watchdog.py         # 验证码处理
│       ├── popups_watchdog.py          # 弹窗处理
│       ├── permissions_watchdog.py     # 权限处理
│       ├── recording_watchdog.py       # 录制
│       └── storage_state_watchdog.py   # 存储状态
├── Agents/                     # Agent 层
│   ├── core/                   # 核心：主循环、工具注册、提示词、运行时
│   │   ├── powerapps_mvp_agent.py  # 主 Agent 入口
│   │   ├── tools.py                # 工具注册
│   │   ├── prompts.py              # 系统提示词
│   │   ├── runtime.py              # 浏览器运行时适配
│   │   ├── doc_extract.py          # 文档记忆提取
│   │   └── project_commands.py     # /project 命令解析
│   ├── memory/                 # 记忆模块
│   │   ├── memory.py           # 项目持久化记忆
│   │   └── short_memory.py     # 短期对话记忆
│   ├── config/                 # 配置
│   │   └── env.py              # 环境变量加载
│   └── LLM/                    # LLM 客户端
│       └── deepseek_client.py  # DeepSeek API 封装
├── MocProcess/                 # PowerApps 自动化操作
│   ├── actions/                # 原子操作，每个文件只能有一个原子操作
│   │   ├── insert_menu.py      # 插入菜单操作
│   │   ├── formula_bar.py      # 公式栏/属性选择器操作
│   │   └── click_funcInput.py  # 公式编辑器聚焦+写入
│   ├── chains/                 # 流程编排
│   │   ├── insert_and_set_formula.py  # 插入控件+设置属性
│   │   └── set_property_formula.py    # 修改已有控件属性
│   └── results/                # 操作结果报告
└── DOM/                        # DOM 解析层
    ├── service.py              # DOM 服务
    ├── views.py                # DOM 数据模型
    ├── enhanced_snapshot.py    # 增强快照
    ├── utils.py                # DOM 工具
    └── serializer/             # DOM 序列化
```
